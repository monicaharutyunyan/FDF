# FDF
"Fil de Fer", my first graphical project (got 100/125).
 

### How to compile
```
git clone https://github.com/monicaharutyunyan/FDF.git
cd fdf
make 
```

### How to use
./fdf <map_file>
Exemple:
`./fdf ./MGDS_WHOLE_WORLD_OCEAN1_XXL.fdf`
